# ✨ Lesson 2：基本型別 (Basic Types)

| 類型                 | 範例                            | 說明       |
| -------------------- | ------------------------------- | ---------- |
| `string`             | `"Hello"`                       | 文字字串   |
| `number`             | `123`, `3.14`                   | 數字       |
| `boolean`            | `true`, `false`                 | 布林值     |
| `any`                | `任意型別`                      | 不建議使用 |
| `array`              | `number[]`、`string[]`          | 陣列       |
| `object`             | `{ name: string, age: number }` | 物件       |
| `undefined` / `null` |                                 | 特殊狀態   |
